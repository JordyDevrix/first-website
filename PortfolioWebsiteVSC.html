Wie ben ik?

Ik ben Jordy de Vries. Ik ben geboren in Den Haag en woon sinds mijn 2e in Wassenaar. Mijn hobby's zijn rommelen met computers en arduino's, music producen en gamen. Alle drie doe ik graag in mijn vrije tijd.

Vooropleiding

Ik heb op het Adelbert College in Wassenaar N&G op het HAVO gedaan, waarna ik heb ingeschreven voor Mechatronica bij De Haagse hogeschool in Delft. Dat ging echter niet zoals gehoopt. De opleiding was niet zoals ik had verwacht. Ik wilde eigenlijk meer met software en ICT werken en zelf programma's schrijven wat op de opleiding nihil gegeven werd. Ik vond het programmeren wel leuk dus ik heb daarom besloten om in te schrijven voor informatica.

Ervaring

Ik heb al enige ervaring met Java en C. Ik heb op de middelbare heel kort een paar gastlessen gekregen over Java waarin we zelf uiteindelijk als eindopdracht een game moesten programmeren. De stof die we in de gastlessen kregen was wel gelimiteerd dus ik zocht meestal dingen op internet. Verder heb ik tijdens Mechatronica geleerd Console applications te maken en AVR's te programmeren. Ook heb ik daar geleerd om te gaan met AVR's en de Datasheets af te lezen. informatica bestaat natuurlijk niet alleen uit programmeren, maar ook versiebeheer speelt een belangrijke rol als het gaat om software development. Ik heb kort de basis van Git geleerd en hoe ik GitHub kan gebruiken om in een team aan één project te werken.

Leeftijdcontrole

Ik heb zelf een leeftijdcontrole app gemaakt waarmee snel een leeftijd controle kan worden uitgevoerd. Dit is bijvoorbeeld handig op plekken waar een minimum leeftijd van 18 jaar is verreist. Dit systeem wordt al volop in supermarkten gebruikt. Ik had de app gemaakt om mijn ervaring te testen in C.

Studs converter

Dit was een kleine applicatie om true size game development in roblox "makkelijker" te maken. Je kan met deze app makkelijk van metric naar studs en vice versa.

AVR Control unit

Op mijn vorige opleiding had ik de code voor een 'tilhulp' geprogrammeerd. De tilhulp was verbonden aan een bedieningspaneel en moest semi-automatisch steigerdelen op elkaar plaatsen.

AVR PWM

PWM staat voor Pulse Width Modulation. Het is een manier om een uitgangspanning in voltage hoger of lager te maken of zelfse te laten wisselen zoals een sinus of blokgolf.

Clock

Ik heb in Java een clock gemaakt. Yup.

Game

Ik heb voor een school opdracht een game gemaakt in Java. De game was een soort arcade game waarbij je steeds tussen twee lijnen moest blijven.

Kennis

Ik kan momenteel C en een klein beetje Java, en dankzij google en een goede vriend ben ik nu ook HTML en CSS aan het leren. Bij HTML en CSS zit ik alleen nu nog op het puntje van de ijsberg. Ik hoop dan ook nog veel meer over HTML en CSS te leren. En natuurlijk ook andere talen die mij kunnen helpen in wat ik wil bereiken.
